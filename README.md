# Sistema de Biblioteca Django

Aplicación web desarrollada en Django para la gestión de libros, usuarios y préstamos.

## Funcionalidades

- CRUD de libros
- CRUD de usuarios
- Gestión de préstamos
- Control de disponibilidad de libros

## Tecnologías

- Python
- Django
- SQLite
- Bootstrap
